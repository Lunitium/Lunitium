# Lunitium

![NuGet Version](https://img.shields.io/nuget/v/Lunitium.DependencyInjection?style=for-the-badge&link=https%3A%2F%2Fwww.nuget.org%2Fpackages%2FLunitium.DependencyInjection%2F)
![GitHub Actions Workflow Status](https://img.shields.io/github/actions/workflow/status/Lunitium/Lunitium/build.yml?style=for-the-badge)

Lunitium is an ecosystem of libraries designed to refine the developer experience by prioritizing ergonomics and drastically reducing boilerplate code.

## Libs

- [Lunitium.DependencyInjection](https://github.com/Lunitium/Lunitium/wiki/Lunitium.DependencyInjection) A fluent and lightweight approach to managing service lifecycles without the usual ceremony.
  - Generate function to register all services
  - All lifetime supported
  - Keyed supported
  - Factory supported

```shell
dotnet add package Lunitium.DependencyInjection
```

```cs
[Dependency]
public class Service
{
}

// <auto-generated />
public static class DependencyInjectionExtensions
{
    public static IServiceCollection AddLunitiumDependencies(this IServiceCollection services)
    {
        services.AddScoped<Lunitium.DependencyInjection.Test.Services.Service>();
        return services;
    }
}

// Usage
var builder = new ServiceCollection();
builder.AddLunitiumDependencies();
var serviceProvider = builder.BuildServiceProvider();
var service = serviceProvider.GetRequiredService<Service>();
```
